A. 基础衍生特征14 个

从主表字段直接计算，不需要关联其他表。

债务负担类

特征计算公式业务含义
CREDIT_INCOME_RATIOAMT_CREDIT / (AMT_INCOME_TOTAL + 1)贷款金额 / 年收入。值=5 表示贷款是年收入的 5 倍
ANNUITY_INCOME_RATIOAMT_ANNUITY / (AMT_INCOME_TOTAL + 1)年还款 / 年收入。>0.5 说明超过一半收入用于还贷

年龄与工龄

特征计算公式业务含义
AGE_YEARS-DAYS_BIRTH / 365申请时年龄
AGE_GROUPpd.cut(AGE_YEARS, [0,25,35,45,55,100])年龄分段：0=<25，1=25-35，2=35-45，3=45-55，4=55+
EMPLOYMENT_YEARS-DAYS_EMPLOYED / 365当前工作年限
INCOME_PER_EMPLOYMENT_YEARAMT_INCOME / (EMPLOYMENT_YEARS + 1)每工龄年收入，反映职业成长速度
CHILDREN_RATIOCNT_CHILDREN / (CNT_FAM_MEMBERS + 1)子女占家庭比例，值高说明负担重

外部评分（最强预测力）

特征计算公式业务含义
EXT_SOURCE_MEANmean(EXT_SOURCE_1, 2, 3)IV=0.63 最强 三个外部征信评分均值
EXT_SOURCE_STDstd(EXT_SOURCE_1, 2, 3)评分标准差，值高说明征信机构评价分歧大
EXT_SOURCE_MINmin(EXT_SOURCE_1, 2, 3)最低评分，"木桶效应"
EXT_SOURCE_MAXmax(EXT_SOURCE_1, 2, 3)最高评分
DOCUMENT_COUNTsum(FLAG_DOCUMENT_*)提交文件数量
IS_RETIRED(DAYS_EMPLOYED == 365243) → 0/1退休标志（365243 是数据中的特殊值）

数据处理补充说明

编码方式

类别数量编码方法例子
2 种Label Encoding → 0/1性别 M/F
3-10 种One-Hot Encoding教育程度
11+ 种Frequency Encoding（默认）用该类别出现比例替代

缺失值处理

缺失率 > 80% 的列 → 直接删除
数值列 → 按 NAME_INCOME_TYPE 分组中位数填充 → 全局中位数兜底 → 0 填充
类别列 → 众数填充

高相关特征删除

Pearson 相关系数 > 0.95 的特征对中删除后出现的那个。从 277 个特征删到 222 个。
